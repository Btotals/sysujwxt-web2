sysujwxt-web
============

python实现的SYSU教务第三方查询网站

部署
----

### 依赖

* python2
* flask
* pycurl
* fake-sysujwxt

### 访问

支持浏览器为:

* Firefox
* Chrome
* safari
* IE8+
* opera
* UC（及主流的手机浏览器）

部署好后访问[http://localhost:5000](http://localhost:5000)即可。
